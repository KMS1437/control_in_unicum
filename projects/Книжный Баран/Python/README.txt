# КнижныйБаран (Bookstore) - Консольный магазин книг

## Описание
Приветствуем вас в Книжном магазине "КнижныйБаран"! Это консольное приложение представляет собой простой интерфейс для покупки книг.

## Использование
1. Запустите программу, выполнив `python your_file_name.py` в командной строке.
2. Вас приветствует сообщение о приветствии и выводится меню с доступными книгами.
3. Выберите книгу, введя соответствующий номер. Можете также ввести "Оплатить" для завершения заказа или "Удалить" для удаления позиции из заказа.
4. Если выбрана оплата, укажите способ оплаты: "1" для наличных, "2" для карты.
5. При оплате наличными, получите книги по указанному адресу. При оплате картой, введите номер карты, дату и 3-значный код на обратной стороне.
6. Вам будет предоставлена квитанция с общей суммой заказа и списком купленных книг.

## Структура кода
- `Bookstore` - класс, описывающий магазин, его меню и заказ.
  - `__init__`: Инициализация магазина с предустановленным меню и пустым заказом.
  - `welcome_message`: Вывод приветствия и текущего меню.
  - `process_order`: Обработка заказа пользователя.
  - `checkout`: Оплата заказа, выбор способа оплаты.
  - `pay_cash`: Оплата наличными.
  - `pay_card`: Оплата картой.
  - `show_order_summary`: Вывод краткого обзора заказа и общей суммы.
  - `calculate_total`: Расчет общей суммы заказа.
  - `remove_item`: Удаление выбранной книги из заказа.
  - `add_item`: Добавление выбранной книги в заказ.

## Пример использования
```python
if __name__ == "__main__":
    bookstore = Bookstore()
    bookstore.welcome_message()
    bookstore.process_order()
```
